## Zadanie 0.1 - Elo Żelo
Zainstalujcie node js i napiszcie prosty skrypt, który wypisze do pliku i konsoli "Elo żelo.txt" frazę "Elo żelo" w osobnych linijkach X razy. X jest ilością minut danej godziny, w której skrypt został uruchomiony.
